# Launch project

- Frontend:
install the dependencies and run in dev mode
```shell
npm i
npm run dev
```

- Backend:
install the dependencies and run in dev mode
```shell
npm i
npm run start:dev
```

Go to [http://localhost:5173](http://localhost:5173) to test the app

## Test

Tests can be run in the frontend or backend
```shell
npm run test
```
